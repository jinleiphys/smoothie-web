Overview

Modern Fortran 95 code for nuclear physics calculations

SMOOTHIE is written in modern Fortran 95 and designed for high-performance nuclear physics calculations. The code implements the Ichimura-Austern-Vincent formalism for non-elastic breakup reactions and uses advanced numerical methods including the Lagrange-mesh R-matrix approach.

üîß Troubleshooting

‚ùå Compilation Errors

Problem: "Command not found: gfortran"

Solution: Install Fortran compiler package for your system.

Problem: "Cannot find -llapack"

Solution: Install LAPACK development libraries or modify library paths in make.inc.

Problem: Module compilation errors

Solution: Ensure Fortran 95 or later compiler. Clean and rebuild all modules.

‚ö†Ô∏è Runtime Issues

Problem: Segmentation fault during execution

Solution: Increase stack size or compile with bounds checking for debugging.

Problem: Numerical instabilities

Solution: Adjust mesh parameters or potential model settings in input file.

Problem: Slow performance

Solution: Link with optimized BLAS/LAPACK libraries and enable compiler optimizations.

üîç Platform-Specific Issues

macOS: Use Homebrew gcc instead of Xcode's compiler for better Fortran support.

Windows: Use MinGW-w64 or Windows Subsystem for Linux (WSL) for best compatibility.

HPC Systems: Load appropriate modules for compiler and mathematical libraries.

‚ö° Performance Optimization

üöÄ Compiler Optimizations

  • Use -O3 or -Ofast for production runs
  • Enable vectorization with -ftree-vectorize
  • Consider profile-guided optimization (PGO)
  • Use native architecture flags: -march=native

üìö Library Optimizations

  • Link with optimized BLAS: Intel MKL, OpenBLAS, or ATLAS
  • Use threaded libraries for parallel execution
  • Set environment variables: OMP_NUM_THREADS
  • Consider GPU acceleration where available

üéõÔ∏è Runtime Tuning

  • Adjust mesh parameters for accuracy vs. speed trade-off
  • Optimize angular momentum cutoffs
  • Use appropriate numerical integration methods
  • Monitor memory usage and adjust accordingly
